Discord bot developed in Node.JS for my private server.
Based around using market API's for Black Desert Online.

Mainly used to test programming in Node.JS, exploration with API's and data along with just general playing with other features. 

Features:
BDO Market API search function
Automatic translator using google translate
Auto role on join
Role on reaction
Ability to store market data in spreadsheet format
BDO Tax calculator
BDO relic/pila/voodoo calculator
